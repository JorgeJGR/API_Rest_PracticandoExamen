API Rest EMPLEADOS - MySQL Server

Database bdempleados
Relación Departamento (1) —— (N) Empleado
Content-Type application/json
Accept application/json


Recursos: DEPARTAMENTOS

MÉTODO PATH DESCRIPCIÓN
GET /apirestjpa/empleados/departamentos Obtiene todos los departamentos
GET /apirestjpa/empleados/departamentos/{id}
/apirestjpa/empleados/departamentos/1 		Obtiene un departamento por ID
POST /apirestjpa/empleados/departamentos Inserta un nuevo departamento
PUT /apirestjpa/empleados/departamentos Actualiza un departamento existente
DELETE /apirestjpa/empleados/departamentos/{id} Elimina un departamento


Recursos: EMPLEADOS

MÉTODO PATH DESCRIPCIÓN
GET /apirestjpa/empleados/empleados Obtiene todos los empleados
GET /apirestjpa/empleados/empleados/{id}
/apirestjpa/empleados/empleados/2 		Obtiene un empleado por ID
POST /apirestjpa/empleados/empleados Inserta un nuevo empleado
PUT /apirestjpa/empleados/empleados Actualiza un empleado
DELETE /apirestjpa/empleados/empleados/{id} Elimina un empleado


Consultas Avanzadas (JPQL / NamedQuery)

TIPO PATH DESCRIPCIÓN
JPQL /apirestjpa/empleados/empleados/departamento/{id} Obtiene los empleados de un departamento (JPQL)
NamedQuery /apirestjpa/empleados/empleados/email/{email} Busca un empleado por email (NamedQuery)
NamedQuery /apirestjpa/empleados/departamentos/nombre/{nombre} Busca un departamento por nombre (NamedQuery)
JPQL /apirestjpa/empleados/empleados/apellidos/{texto} Obtiene empleados cuyo apellido contenga el texto indicado (JPQL)
JPQL /apirestjpa/empleados/departamentos/con-empleados Obtiene los departamentos que tienen al menos un empleado (JPQL)
NamedQuery /apirestjpa/empleados/empleados/telefono/{telefono} Busca un empleado por teléfono exacto (NamedQuery)
NamedQuery /apirestjpa/empleados/departamentos/sin-empleados Obtiene los departamentos que no tienen empleados (NamedQuery)
